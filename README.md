## Demo Flow

1. Open the dashboard.
2. Create or load a road project.
3. Run AI estimation.
4. Review prediction outputs.
5. Compare multiple projects.
6. Generate PDF and Excel reports.
7. Configure app settings.

## Core Outputs

- Total project cost
- Construction duration
- Material index
- Manpower hours per km
- Machinery hours per km
- Cost breakdown charts
- Resource intensity charts
- Feature importance
- Project comparison
- Downloadable reports
